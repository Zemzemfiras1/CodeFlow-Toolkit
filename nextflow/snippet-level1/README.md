# Snippet Level 1 — Basics

This level introduces the **fundamentals of Nextflow**:

- **1-hello.nf** → Minimal "Hello World" pipeline.
- **2-countlines.nf** → Count the number of lines in a file.
- **3-python.nf** → Run a Python script inside a Nextflow process.
- **4-directives.nf** → Example of using basic process directives.
- **5-tainingHello.nf** → Variant of the Hello World workflow.
- **file.txt** → Example input file for some processes.

> ⚠️ **Note:** These pipelines are simplified examples for **teaching purposes**.

➡️ Start here if you are new to Nextflow. Learn how to define a process, run commands, and handle simple inputs.
